Installation
------------

Installer creates symbolic links in your `$HOME`.  It asks whether
to backup files that otherwise would be replaced.  Requires Python
and [Homebrew][].

  [Homebrew]: https://brew.sh


### Fresh

    cd ~
    git clone https://github.com/narfdotpl/dotfiles.git
    cd dotfiles
    ./install


### Update

    h
    /
    .
    g m
    ./install
